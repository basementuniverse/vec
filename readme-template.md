# vec

A small vector and matrix library

## Usage

Node:

```javascript
const { vec, mat } = require('@basementuniverse/vec');
```

Browser:

```html
<script src="vec.js"></script>
```

Typescript:

```typescript
import { vec, mat } from '@basementuniverse/vec';
```
